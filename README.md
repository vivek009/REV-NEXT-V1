# Catalog Manager

The **Catalog Manager** system provides end-to-end management of product configuration, pricing, and quotation.  
It is designed to simplify catalog operations, ensure pricing accuracy, and streamline quote generation for customers.

---

## Features

- **Configure**: Define and customize products, attributes, and rules.
- **Price**: Apply pricing models, discounts, and adjustments in real-time.
- **Quote**: Generate professional, shareable quotes for customers.

---

## Flow

The system follows a **CPQ (Configure → Price → Quote)** flow
